# 사각사각 서비스 디자인 계약

## 방향

사각사각은 공개 업로드 서비스가 아니라, 편집자가 선별한 PDF 문집을 독자가 읽는 밝은 서재다. 화면의 가장 큰 면적은 원본 PDF에 양보하고 UI는 얇게 유지한다.

## 화면

- 서재와 뷰어만 제공한다.
- 데스크톱에서는 PDF 두 쪽을 펼친다.
- 태블릿과 모바일에서는 한 쪽을 크게 보여준다.
- 상단에는 브랜드와 서재·뷰어 이동만 둔다.
- 하단 도구에는 페이지 이동, 이미지 스티커, 현재 쪽 저장만 둔다.

## 스티커

- 스티커는 문구를 직접 렌더링하지 않는다.
- `public/stickers/`의 SVG 애셋을 버튼과 페이지 위에 함께 사용한다.
- 페이지 위 스티커는 각 PDF 쪽의 좌표를 기준으로 이동한다.

## PDF 등록

- 공개 파일 업로드 UI를 제공하지 않는다.
- 편집자가 관리하는 PDF만 `public/books/`와 `manifest.json`에 등록한다.
- 원본 PDF의 페이지 순서와 디자인은 PDF.js로 그대로 렌더링한다.
